# COM_IE

**Category:** User Input
**Indexes:** r, datayear, c, s

## Description

Infrastructure or transmission efficiency of a commodity.

## Details

Direct inheritance. Weighted aggregation.

## Units and Defaults

Decimal fraction (0,∞); default value: 1; Default i/e: STD

## Related Parameters and Sets

None documented.

## Affected Equations and Variables

Overall efficiency applied to the total production of a commodity in the commodity balance equation (EQ(l)_COMBAL).
