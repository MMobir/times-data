# PAR_COMPRDL

**Category:** Reporting
**Indexes:** r, t, c, s

## Description

Commodity Total Production: Level value of the commodity production variable (VAR_COMPRD). The variable represents the total production of a commodity. It is only reported, if a bound or cost is specified for it or it is used in a user constraint.
