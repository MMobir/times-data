# NCAP_FSUBM

**Category:** User Input
**Indexes:** r, datayear, p

## Description

Period sensitive multiplier curve (MULTI) applied to the subsidy (NCAP_FSUB).

## Details

Provided when shaping based upon the period is desired.

## Units and Defaults

Integer number; Default value: 0; Default extrapolation: MIG
