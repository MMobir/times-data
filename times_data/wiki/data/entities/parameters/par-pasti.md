# PAR_PASTI

**Category:** Reporting
**Indexes:** r, t, p, v

## Description

Technology Capacity: Residual capacity of past investments (NCAP_PASTI) of process (p) still existing in period (t), where vintage (v) is set to '0' to distinguish residual capacity from new capacity.
