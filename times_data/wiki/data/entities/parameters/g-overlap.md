# G_OVERLAP

**Category:** User Input
**Indexes:** 

## Description

Overlap of stepped solutions (in years).

## Details

Used only when time-stepped solution is activated with the TIMESTEP control variable.

## Units and Defaults

Scalar [0,100]; Default value: TIMESTEP/2

None documented.
