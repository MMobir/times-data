# PAR_EOUT

**Category:** Reporting
**Indexes:** r, v, t, p, c

## Description

Electricity supply by technology and energy source (optional): Electricity output of electricity supply processes by energy source; based on using NRG_TMAP to identify electricity commodities, but excludes standard and storage processes having electricity as input. (Opted out by default – set RPT_OPT('FLO','5')=1 to activate; see Part III, Section 3.10).
