# PAR_CAPL

**Category:** Reporting
**Indexes:** r, t, p

## Description

Technology Capacity: Capacity of process (p) in period (t), derived from VAR_NCAP in previous periods summed over all vintage periods. For still existing past investments, see PAR_PASTI.
