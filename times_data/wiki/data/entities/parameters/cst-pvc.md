# CST_PVC

**Category:** Reporting
**Indexes:** uc_n, r, c

## Description

Total discounted costs by commodity (optional, activate by setting RPT_OPT('OBJ','1')=1): Total present value of commodity-related costs in the base year, by type (with types COM, ELS, DAM). See Part III, Section 3.10 on the reporting options, and Table 16 below for acronym explanations.
