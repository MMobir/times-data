# CST_FLOX

**Category:** Reporting
**Indexes:** r, v, t, p, c

## Description

Annual flow taxes/subsidies: Annual undiscounted flow related costs (caused by FLO_TAX, FLO_SUB) in period (t) associated with a commodity (c) flow in/out of a process (p) with vintage period (v) as well as capacity related commodity flows (specified by NCAP_COM, NCAP_ICOM, NCAP_OCOM).
