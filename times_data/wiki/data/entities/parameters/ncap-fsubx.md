# NCAP_FSUBX

**Category:** User Input
**Indexes:** r, datayear, p

## Description

Age-based shaping curve (SHAPE) to be applied to the fixed subsidy (NCAP_FSUB).

## Details

Provided when shaping based upon age is desired.

## Units and Defaults

Integer number; Default value: 0; Default extrapolation: MIG
