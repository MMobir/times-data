# CM_MAXC

**Category:** User Input
**Indexes:** datayear, item

## Description

Maximum level of climate variable

## Details

Since no default inter-/extrapolation, bounds must be explicitly specified for each desired year, unless an explicit inter-/extrapolation option is set. See Appendix on Climate Module for details.

## Units and Defaults

Climate variable unit [0,∞); default value: none

## Related Parameters and Sets

None documented.
