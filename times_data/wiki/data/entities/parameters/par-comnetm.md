# PAR_COMNETM

**Category:** Reporting
**Indexes:** r, t, c, s

## Description

Commodity Net – Marginal: Undiscounted annual reduced costs of the VAR_COMNET variable of commodity (c). It is only reported, if a bound or cost is specified for it or it is used in a user constraint.

No additional details available.
