# UC_CLI

**Category:** User Input
**Indexes:** uc_n, side, r, datayear, item

## Description

Multiplier of climate variable in user constraint

## Details

Used in user constraints. Climate variable can be at least any of CO2-GTC, CO2-ATM, CO2-UP, CO2-LO, FORCING, DELTA-ATM, DELTA-LO (for carbon). See Appendix on Climate Module for details.

## Units and Defaults

Dimensionless [open]; default value: none; Default i/e: STD

## Related Parameters and Sets

None documented.
