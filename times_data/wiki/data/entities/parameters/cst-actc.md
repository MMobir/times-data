# CST_ACTC

**Category:** Reporting
**Indexes:** r, v, t, p, uc_n

## Description

Annual activity costs: Annual undiscounted variable costs (caused by ACT_COST) in period (t) associated with the operation (activity) of a process (p) with vintage period (v). Additional indicator (uc_n) for start-up costs.
