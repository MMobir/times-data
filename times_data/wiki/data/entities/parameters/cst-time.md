# CST_TIME

**Category:** Reporting
**Indexes:** r, t, s, uc_n

## Description

Discounted value of time by period: Present value of the time in each model period (t) by region (r), with s='ANNUAL' and uc_n='COST'/'LEVCOST' depending on whether the $SET ANNCOST LEV reporting option has been used.
