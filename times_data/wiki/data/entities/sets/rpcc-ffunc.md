# rpcc_ffunc (r,p,c)

**Category:** Internal
**Aliases:** None

## Description

Flow variable of a commodity (c) associated with a process (p) that can be replaced by another flow variable of the process, due to a direct FLO_FUNC or FLO_SUM relationship.

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
