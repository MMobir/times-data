# top (r,p,c,io)

**Category:** User Input
**Aliases:** None

## Description

RES topology definition indicating that commodity c enters (io=’IN’) or leaves (io='OUT') the process p [set of quadruples {r,p,c,io} such that process p has a flow of commodity c with orientation io in region r].

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
