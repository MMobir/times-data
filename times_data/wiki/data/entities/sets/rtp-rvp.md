# rtp = rvp (r,v,p)

**Category:** Internal
**Aliases:** None

## Description

Indication of the periods and pastyears for which process (p) in region (r) is available; all other RTP_* control sets are based on this set.

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
