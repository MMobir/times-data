# nrg_type

**Category:** Internal
**Aliases:** None

## Description

List of energy types; internally established in MAPLIST.DEF as: FOSSIL, NUCLR, SYNTH, FRERENEW, LIMRENEW, ELC, HTHEAT, LTHEAT, CONSRV. The predefined elements should not be deleted.
