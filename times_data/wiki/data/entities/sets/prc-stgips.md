# prc_stgips (r,p,c)

**Category:** Internal
**Aliases:** None

## Description

Set of triplets {r,p,c} such that process p is an inter-period storage for the commodity c in region r; the commodity c enters and/or leaves the storage according to the set top; the storage can only operate at the ANNUAL level.

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
