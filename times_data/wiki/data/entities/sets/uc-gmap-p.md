# uc_gmap_p (r,uc_n,uc_grptype,p)

**Category:** User Input
**Aliases:** None

## Description

Indicator that a variable (VAR_ACT, VAR_NCAP or VAR_CAP) associated with a process (p) in a region (r) appears in a user constraint (uc_n).
