# prc_ts (all_r,p,s)

**Category:** User Input
**Aliases:** prc_ts2

## Description

Set of triplets {all_r,p,s} such that process p can operate at timeslice s in region r; Default: all timeslices on the timeslice level specified by prc_tsl.

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
