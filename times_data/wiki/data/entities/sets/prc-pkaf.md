# prc_pkaf (all_r,p)

**Category:** User Input
**Aliases:** None

## Description

Set of pairs {all_r,p} such that the availability factor (ncap_af) is to be used as value for the fraction of capacity of process p that can contribute to the peaking constraints (ncap_pkcnt), in region r.

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
