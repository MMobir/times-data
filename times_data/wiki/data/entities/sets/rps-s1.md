# rps_s1 (r,p,s)

**Category:** Internal
**Aliases:** None

## Description

All (permitted) timeslices (s) belonging to the finest timeslice level of the process (p, prc_tsl) and the commodity timeslice level (com_tsl) of the shadow primary commodity group.

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
