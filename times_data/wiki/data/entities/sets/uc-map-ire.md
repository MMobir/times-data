# uc_map_ire (uc_n,r,p,c)

**Category:** User Input
**Aliases:** None

## Description

Indicator that an import/export (according to top_ire) trade variable (VAR_IRE) for region r, process p, and commodity c is involved in a user constraint (uc_n).

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
