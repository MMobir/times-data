# prc_dscncap (r,p)

**Category:** User Input
**Aliases:** None

## Description

Set of processes p to be modelled using the lumpy investment formulation in region r; Default: empty set. If p is not in this set, then any lumpy investment parameters provided for p are ignored.

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
