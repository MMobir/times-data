# pastyear (year)

**Category:** User Input
**Aliases:** pyr

## Description

Years for which past investments are specified; pastyears usually lie before the beginning of the first period; No default.

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
