# prc_map (r,prc_grp,p)

**Category:** User Input
**Aliases:** None

## Description

Grouping of processes into process groups (prc_grp) [set of triplets {r,prc_grp,p} such that process p belongs to group prc_grp in region r]. Note: used strictly for reporting purposes.
