# rps_s2 (r,p,s)

**Category:** Internal
**Aliases:** None

## Description

For an ANNUAL level NST process, contains all permitted timeslices (s) at the level above the finest commodity timeslice levels (com_tsl) of the shadow primary group (spg). For all other processes, rps_s2 = rps_s1.

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
