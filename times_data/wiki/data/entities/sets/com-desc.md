# com_desc (r,c)

**Category:** User Input
**Aliases:** None

## Description

Commodities by region, only to facilitate different descriptions by region. The elements are pairs {r,c}, for which the description is specified according to the GAMS syntax.

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
