# prc_tsl (r,p,tslvl)

**Category:** User Input
**Aliases:** None

## Description

Set of triplets {r,p,tslvl} such that process p can operate at timeslice level tslvl in region r; Default: 'ANNUAL' timeslice level.

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
