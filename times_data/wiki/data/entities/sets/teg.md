# teg (p)

**Category:** User Input
**Aliases:** None

## Description

Set of technologies selected for endogenous technology learning; Subset of set p; if p not in teg, then any ETL investment parameters provided are ignored.

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
