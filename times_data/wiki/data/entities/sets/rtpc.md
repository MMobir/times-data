# rtpc (r,t,p,c)

**Category:** Internal
**Aliases:** None

## Description

For a process (p) in a region (r) the combination of the periods it is available (rtp) and commodities associated with it (rpc).

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
