# eachyear (year)

**Category:** Internal
**Aliases:** None

## Description

Set of all years between scalars MINYR (first year needed for cost calculation in objective function) and MIYR_VL + DUR_MAX (estimation of last year possible cost terms may occur).

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
