# uc_gmap_c (r,uc_n,uc_grptype,c)

**Category:** User Input
**Aliases:** None

## Description

Indicator that a commodity variable (VAR_COMCON or VAR_COMPRD) for commodity (c) in a region (r) appears in a user constraint (uc_n).
