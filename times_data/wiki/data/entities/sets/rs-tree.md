# rs_tree (all_r,ts,s)

**Category:** Internal
**Aliases:** None

## Description

For a timeslice (ts) all timeslices (s) that are on the same paths within the timeslice tree, e.g. if ts=SP_WD in Fig. 6, valid timeslices s are: ANNUAL, SP, SP_WD, SP_WD_D, SP_WD_N

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
