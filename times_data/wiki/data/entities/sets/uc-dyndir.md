# uc_dyndir (r,uc_n,side)

**Category:** User Input
**Aliases:** None

## Description

If side = 'RHS', indicator for growth constraints to be generated between the periods t–1 and t; if side = 'LHS', the set is ignored.

## Usage Notes

See the [Sets chapter](../../raw/docs/part-2/02-sets.md) in the TIMES Reference Manual for full details on how this set is used in the model equations.
