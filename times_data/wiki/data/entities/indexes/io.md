# io

**Aliases:** inout

## Description

Input/Output indicator for defining whether a commodity flow enters or leaves a process; internally fixed = 'IN' for enters and 'OUT' for leaves.

## Related Indexes

None documented.

## Usage

This index is used as a dimension in various TIMES parameters, sets, and equations. See the [Reference Manual Introduction](../../raw/docs/part-2/01-introduction.md) for the full notation conventions.
