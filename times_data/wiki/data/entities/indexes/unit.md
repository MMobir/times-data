# unit

**Aliases:** None

## Description

List of capacity blocks that can be added in lumpy investment option; default elements 0–100; the element '0' describes the case when no capacity is added.

## Related Indexes

None documented.

## Usage

This index is used as a dimension in various TIMES parameters, sets, and equations. See the [Reference Manual Introduction](../../raw/docs/part-2/01-introduction.md) for the full notation conventions.
